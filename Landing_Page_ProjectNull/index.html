Game Overview

Project Null is an immersive space battle game where players command unique planets in strategic combat. The game combines tactical decision-making with unique planet abilities in a turn-based battle system.

Core Gameplay

  • Turn-based strategic combat
  • Multiple planet types with special abilities
  • Resource management and tactical choices
  • Single-player and multiplayer modes
  • Persistent game state with save/load system

Technical Features

Architecture

  • Built with Java Swing
  • SQLite database for save management
  • Object-oriented design
  • Modular architecture
  • Cross-platform compatibility

System Requirements

  • Java Runtime Environment (JRE) 17+
  • SQLite JDBC driver
  • Minimum 4GB RAM
  • 500MB free disk space
  • Windows/Mac/Linux compatible

Game Modes

  • New Game: Custom team setup
  • Load Game: Continue previous battles
  • Single Player: vs AI opponents
  • Multiplayer: Local or online
  • Challenge Mode: Special scenarios
